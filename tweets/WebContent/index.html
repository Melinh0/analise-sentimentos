Análise de Sentimentos: Uma Visão Sobre o Governo de Michel Temer

Este projeto tem por finalidade implementar técnicas de aprendizado de máquina supervisionado, para análise de sentimento em mídias sociais, precisamente no Twitter, sobre o atual Presidente da República, Michel Temer.

Esta ferramenta foi desenvolvida para apoio à coleta de informações, onde um banco de dados de tweets sentimentalizados será desenvolvido. Este dataset será utilizado pelo algoritmo para treino e comparação com posteriores resultados de sentimentalização automatizada.

Desenvolvido por Bruno Pacheco Lopes da Silva, MBA em Engenharia de Software - Escola Politécnica / UFRJ.
