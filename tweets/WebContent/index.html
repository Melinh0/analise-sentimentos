Análise de Sentimentos: Uma Visão Sobre o Governo de Michel Temer

Este projeto tem por finalidade implementar técnicas de aprendizado de máquina, para análise de sentimento. Para tal, serão utilizados tweets relacionados ao atual governo federal, do Presidente Michel Temer. Serão utilizadas técnicas de análise de sentimento baseado em algoritmo de aprendizado de máquina supervisionados.

Esta ferramenta foi desenvolvida para apoio à coleta de informações, para que um banco de dados de tweets sentimentalizados seja desenvolvido, servindo ao algoritmo como dataset de treino, e depois possamos avaliar a acurácia de posteriores resultados.
